Open Nutrition Label Finder on localhost:3000

Features to test:

  • Search for foods (try typing "apple" or "banana")
  • Click on common searches
  • Adjust serving sizes on the nutrition label
  • Visit individual food pages like http://localhost:3000/food/apple

If the app doesn't load:

  1. Make sure the terminal shows "Ready"
  2. Try refreshing the page
  3. Check if any firewall is blocking localhost
